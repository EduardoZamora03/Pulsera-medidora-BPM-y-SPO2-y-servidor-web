# Pulsera-medidora-BPM-y-SPO2-y-servidor-web
Este repositorio muestra un proyecto que emula el funcionamiento de una pulsera para medicina que funciona con 2 ESP32 una funciona como servidor que funciona en todo momento y captura los datos almacenandolos en .CSV y otra es la que recibe los datos y manda esos datos a una pantalla redonda TFT para mostrar los datos obtenidos de BPM y SPO2
